# Правила перевода Gunsaw (для агентов workflow)

Ты переводишь строки игры **Gunsaw** — 2D-шутер про антропоморфных гибридов-животных (наёмники-зверолюди воюют отрядами). Перевод на русский для мода-русификатора (XUnity AutoTranslator). Входные строки — из дампа ассетов, поэтому СРЕДИ НИХ МНОГО НЕ-ТЕКСТА (ID, имена методов, ассетов, кода). Твоя задача: для каждой строки решить вердикт и, если это видимый игроку текст, дать качественный перевод с учётом контекста (тег компонента подсказывает, где строка показывается).

## Вердикты

- **translate** — текст, который игрок ВИДИТ на экране: надписи UI, заголовки, подписи кнопок, описания видов, статы, брифинги миссий, подсказки загрузки, цитаты, типы оружия, строки со счётом («0 kills» → «0 убийств»).
- **skip** — НЕ отображаемый текст, переводить нельзя (сломает игру или это мусор):
  - внутренние ID видов/тегов: `base`, `chik`, `g4a`, `badguy`, `goodguys`, `icke`, `voyager`…
  - имена методов-обработчиков событий: `BackToEditor`, `BloomToggled`, `MasterVolumeChanged`, `UnPause`, `ToggleLighting`
  - состояния UI: `Normal`, `Highlighted`, `Pressed`, `Selected`, `Disabled`
  - имена ассетов/типов/ассембли: `ControlBinder, Assembly-CSharp`, `TMP_FontAsset`-имена, имена шрифтов, `UnityEngine.GameObject, UnityEngine`
  - пути ресурсов: `Sounds/...`, `Spawnables/...`, `Spawnables/Weapons/`
  - плейсхолдеры редактора уровней: `< Panel Title >`, `New Text`, `TEST`, `Vector2`, `Vector4`, `BitField`, `Foldout`, `Color`
  - **инспекторские тултипы для разработчика** (тег часто `LevelPartGame`): технические описания поведения объектов/полей. Признак: «Set this to…», «Determines how…», «A moving tile. Moves when its ID gets activated…», «if false, falling crates wont…». В ИГРУ НЕ ПОПАДАЮТ → skip.
  - **реальные модели оружия** латиницей: `AK47`, `AUG`, `M16`, `MP5`, `Glock`, `FAMAS`, `G36C`, `P90`, `RPG`, `SR25`, `STG`, `Sten`, `Saiga`, `PPSH`, `PDW`, `M1918`, `MP7` — оставить как есть (бренды не переводят) → skip.
  - camelCase имена-идентификаторы оружия: `GrenadeLauncher`, `SawedOff`, `SMGun`, `BigRifle`, `TestGun`, `HeavyRevolver`, `KrisVector`, `MADDeagle`, `TommyGun`, `SnubNose` → skip (это имена ассетов, не подписи; есть дубль-подпись с пробелом).
  - чистая пунктуация / числа без слов / одиночные символы.
- **doubtful** — НЕ уверен, что видно в игре, ИЛИ перевод спорный, ИЛИ нужен контекст, которого нет. Примеры: названия музыкальных треков (`MusicClip`, напр. «50 Million Year Trip»), имена видов с неустоявшимся переводом, двусмысленные короткие строки. Дай свой лучший вариант в `ru`, но пометь `doubtful` и причину в `note`. Эти строки уйдут человеку на проверку.

Если сомневаешься между translate и skip — ставь **doubtful**, не пропускай молча.

## Стиль перевода (translate)

- Верни ТОЛЬКО перевод в поле `ru`. Поле `key` оставь ДОСЛОВНО как во входе.
- **Сохраняй `\n` дословно** (последовательность «обратный слэш + n») — это перенос строки внутри значения. НЕ превращай в реальный перенос.
- **Сохраняй теги** `<color="green">…</color>`, `<color="red">…</color>`, `<i>`, `<I>`, `<b>` дословно — переводи только текст внутри них.
- **Сохраняй маркеры** `- ` в начале строк-списков и пустые строки-отбивки.
- **НИКАКИХ `—` (em-dash) в переводе.** Используй запятую, двоеточие или переформулируй. Обычный дефис `-` разрешён.
- **Опечатки оригинала (В1):** случайные орфо-опечатки причёсывай (`Dissapointing`→«Разочаровываешь», `bizzare`→«странное», `gils`→«жабры», `Narcicissm`→«нарциссизм»). НО сохраняй намеренный нижний регистр, мемы, шутки и каламбуры-неологизмы (`speciwomen` = species+women → русский неологизм «самковидина»).
- **Грубости сохраняй** в силе: `Asshole.`→«Мудак.», ругань не смягчать.

## Тон по типу строки

- **Описания видов** (теги `MainMenuManager`/`BodyScript`, длинные маркированные списки) — голос «составителя бестиария»: нейтрально-описательный, сухой юмор. Заканчиваются тактическим советом.
- **Статы видов** (короткие, `BodyScript`: «Agile and fragile», «Pile of steel\nMissing soul») — телеграфный, ёмкий.
- **Брифинги миссий** (`MissionSelect`) — деловой инструктаж от куратора (подписи «- Z», «- Y» сохранять).
- **Подсказки загрузки** (`SceneLoader~raw`) — инструктивный совет игроку.
- **UI/меню/кнопки** — функциональный, короткий, привычные игровые термины.

## Формат вывода

Запиши результат в `docs/wf_out/out_NNN.json` (NNN = номер твоего батча) — JSON-массив объектов:
```json
[
  {"key": "<дословно ключ из батча>", "verdict": "translate|skip|doubtful", "ru": "<перевод или пусто для skip>", "note": "<кратко: причина для skip/doubtful, иначе пусто>"}
]
```
В JSON последовательность `\n` из ключа должна остаться экранированной (`\\n`). Один объект на каждую входную строку батча — ничего не пропускай.
